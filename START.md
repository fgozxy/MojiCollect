# 🎴 日本語カードゲーム - 启动指南

## 快速开始

### 方法 1: 使用 npm (推荐)
```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 在浏览器中打开 http://localhost:3000
```

### 方法 2: 直接打开
```bash
# 直接在浏览器中打开 index.html 文件
open index.html
```

## 项目特性

✅ **已完成的功能**
- 📱 响应式设计，支持移动端和桌面端
- 🎴 完整的抽卡游戏系统
- 🎵 音频文件支持和管理
- 📚 单词管理系统 (添加、删除、搜索)
- 📊 详细的学习历史记录和统计
- 🤖 自动抽卡模式 (可设置数量和间隔)
- ⚙️ 丰富的设置选项
- 💾 数据导入/导出功能
- 🌙 深色/浅色主题切换

## 游戏模式

### 手动模式
- 点击 "カードを引く" 开始抽卡
- 根据显示的内容输入答案
- 点击 "答え合わせ" 检查答案
- 可跳过或查看正确答案

### 自动模式
- 设置一次抽取的卡片数量
- 设置间隔时间
- 系统自动出题和切换
- 适合连续听写练习

## 出题类型

1. **日本語** - 显示日语，输入假名和中文
2. **仮名** - 显示假名，输入日语和中文
3. **中文** - 显示中文，输入日语和假名
4. **音声** - 播放音频，输入所有内容

## 使用说明

1. **添加单词**: 在 "単語管理" 标签中添加新的日语单词
2. **开始游戏**: 在 "ゲーム" 标签中选择模式并开始
3. **查看进度**: 在 "履歴" 标签中查看学习记录和统计
4. **个性化设置**: 在 "設定" 标签中调整游戏体验

## 技术栈

- **前端**: HTML5, CSS3, Vanilla JavaScript
- **音频**: Web Audio API
- **存储**: LocalStorage
- **样式**: CSS Grid, Flexbox, CSS Variables
- **字体**: Google Fonts (Noto Sans JP)

## 浏览器支持

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 开发

项目采用模块化架构:
- `dataManager.js` - 数据管理
- `audioPlayer.js` - 音频播放
- `gameLogic.js` - 游戏逻辑
- `app.js` - 主应用程序

## 数据存储

所有数据都保存在浏览器的 LocalStorage 中:
- 单词数据
- 学习历史
- 用户设置
- 音频文件 (Base64编码)
